# Phishing Awareness Program

## Objective
This project is an educational presentation created to spread awareness about phishing attacks and how to stay safe online.

## Topics Covered
- What is Phishing?
- Why Phishing is Dangerous
- Types of Phishing
- How to Identify Phishing Emails
- Fake Websites & Login Pages
- Social Engineering Techniques
- Real-Life Phishing Case Study
- Best Practices
- Common Warning Signs
- Quiz
- Conclusion

## Tools Used
- Microsoft PowerPoint
- Internet Research

## Outcome
This presentation helps users identify phishing attempts, understand social engineering techniques, and follow cybersecurity best practices to protect themselves online.
